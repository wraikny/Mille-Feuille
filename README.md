# Mille-Feuille
Mille-Feuille is a Altseed Extension Library for F# 

## Target
.Net Framework 4.7.2

## Build
1. Clone Tart and Affogato to same directory
2. Open Mille-Feuille.sln in Visual Studio 2019 and build.
